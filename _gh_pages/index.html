Bootstrap je nejpopulárnější HTML, CSS, a JS framework pro tvorbu responzivních, mobile-first webových projektů na světě.

Stáhnout Bootstrap

Aktuální verze v4.0.0-alpha.6

Začít je jednoduché

Přímo vložte Bootstrap CSS a JS do jakéhokoliv projektu. Použijte svůj oblíbený správce balíčků, stáhněte si zdrojový kód, nebo použijte Bootstrap CDN.

Řešení závislostí

Připravte Sass a JavaScript soubory přes Bower, Composer, Meteor, nebo npm. Instalace prostřednictvím správce balíčků neobsahují dokumentaci, ale obsahují náš Gruntfile a readme

Bootstrap CDN

Pokud potřebujete pouze vložit předkompilované Bootstrap CSS a JS soubory, použijte Bootstrap CDN, který je k dispozici k volému použití prostřednictvím Max CDN folks.

Vytvořeno pro každého, kdekoliv.

Bootstrap dělá webový vývoj rychlejší a snadnější. Byl vytvořen pro lidi všech schopností, všech možných zařízení, a pro projekty všech velikostí.

Preprocessor

Bootstrap je dodávaný s vanilla CSS, ale jeho zdrojový kód využívá Sass, populární CSS preprocesor. Začnete okamžitě s předpřipraveným CSS nebo svůj projekt postavte na dalších zdrojích.

Jeden framework, všechna zařízení.

Bootstrap jednoduše a efektivně škáluje váš web pomocí jedné jediné kódové základny, od telefonů přes tablety až po desktopy.

Našlapaný funkcemi

S Bootstrapem, získáváte obrovské množství běžných HTML elementů, nesčetné zástupy vlastních HTML a CSS komponent, a skvělé jQuery pluginy.

Bootstrap je open source! Je hostován, vyvíjen a spravován na GitHub lidmi jako jste vy.
